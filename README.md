# Roman Empire Website

A multi-page HTML/CSS website exploring the Roman Empire, including information about its military legions and impressive infrastructure.

## Project Overview

This website is built with HTML and CSS and features three main pages with navigation, styling, and responsive design.

## Pages

- **Home (index.html)** - Overview of the Roman Empire
- **Legions (Legions.html)** - Information about Roman military legions
- **Infrastructure (Roman-infrastructure.html)** - Details about Roman roads, aqueducts, and buildings

## Features

- Multi-page navigation
- External CSS styling
- Responsive design for mobile and desktop
- Images and tables
- Professional color scheme (gold, maroon, and red)

## Files

- `index.html` - Home page
- `Legions.html` - Legions page
- `Roman-infrastructure.html` - Infrastructure page
- `Indexstyle.css` - Styling for home page
- `legionsstyle.css` - Styling for legions page
- `buildings.css` - Styling for infrastructure page
- `SPQR_sign.png` - Roman Empire image

## How to View

Open `index.html` in your web browser to start exploring!
